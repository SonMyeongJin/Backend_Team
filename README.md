## Project Overview

이 프로젝트는 운동을 함께할 사람들을 모아주는 커뮤니티 플랫폼입니다. 사용자는 소셜링과 크루 기능을 통해 손쉽게 운동 모임을 조직하고 관리할 수 있습니다. 더 나아가, 서울시 체육 시설 정보는 공공데이터를 활용하여 제공되며, 모임 장소 선택이 더욱 편리해졌습니다. 이 플랫폼을 통해 사용자들은 더 활기차고 연결된 운동 경험을 누릴 수 있습니다. 회원가입또한 카카오톡 기반으로 하여 가입이 간편합니다.
